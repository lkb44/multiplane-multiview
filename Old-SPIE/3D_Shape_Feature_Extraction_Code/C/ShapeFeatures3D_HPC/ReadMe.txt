
Run script as <Executable> <LabelImage> <IntensityImage> <FeatureValueFile.txt> 

Output feature file has values in the following order:
-----------------------------------------------------
Label: 
Volume:1           
Integrated Intensity (sum within label): 1
Centroid (normalized by voxel-count): 3
Weighted Centroid (normalized by sum of intensities):3 
Axes Length (all axes):3 
MajorAxisLength:1 
MinorAxisLength:1 
Eccentricity:1
Elongation (Fraction of Major and Minor Axes):1           
Orientation (in radians):1 
Bounding box: [xmin xmax ymin ymax zmin zmax]:6
PrincipalMoments (Eigen values):3
Perimeter (surface for 3D):1
Roundness:1
EquivalentSphericalRadius:1
EquivalentSphericalPerimeter (surface):1
EquivalentEllipsoidDiameter:3
Flatness (ratio of first and second moments):1
Elongation Shape Factor (ratio two second moments):1
NumberOfPixelsOnBorder (only if label touches border):1
PerimeterOnBorder (only if label touches border):1
PerimeterOnBorderRatio (only if label touches border):1
Tumor Compactness (Volume/Surface Area):1
------------------------------------------------------
